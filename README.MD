# List Files in Folders

A Python script that accepts folder paths from the user at runtime and lists all files inside each folder. Demonstrates lists, loops, the `os` module, and exception handling.

## How to Run
```bash
python3 list_files_in_folders.py
```

Enter folder paths separated by spaces when prompted.

## Concepts Used
- `input()` and `split()` for runtime user input
- `os.listdir()` to interact with the filesystem
- `for` loops for iteration
- `try/except` for graceful error handling
- `main()` function structure
